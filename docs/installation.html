Installation¶

After downloading the latest distribution, the simplest method is to install via pip:

pip install flyqma-0.1.0-beta.tar.gz

Dependencies¶

  • Python 3.6+

Required:

  • PyTables
  • Scipy
  • Pandas
  • PIL
  • TiffFile
  • Matplotlib
  • Seaborn
  • Scikit Image
  • Scikit Learn
  • Statsmodels
  • NetworkX
  • Infomap
